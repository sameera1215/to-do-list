# To-Do List

## 📖 Description
This is a simple To-Do List application built using Python. It helps users manage their daily tasks.

## ✨ Features
- Add Task
- View Tasks
- Remove Task
- Exit Program

## 💻 Technologies Used
- Python

## 🚀 How to Run
1. Run the todo_list.py file.
2. Choose an option from the menu.
3. Add, view, or remove tasks.
4. Exit the program when finished.
